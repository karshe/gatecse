Day 1
'Set Theory'

“Concept of set appears in all mathematics, we need to know about its terminology and notation.”, he started.

Set can be viewed as any well–defined collection of objects called ‘elements’ or ‘members’ of set. Synonyms for ‘set’ are ‘classes, ‘collection’ and ‘family.’

There essentially two ways of specifying a particular set—one way if possible is to list its members separated by commas and contained in braces {} where–as second way is to state those properties which categorized the elements in set.

He illustrated example for first way as—
A = {‘Bruce’, ‘Will’, ‘Sam’} and another way as
B = {x | x is an integer and x > 0}

Day 1
'Set Theory'

Subsets—suppose every element in set A is also an element of set B then A is called as subset of B. We also say that A is contained in B or B contains A.

All sets under investigation in any application of set theory are assumed to belong to some fixed large set called as ‘universal set’ which is denoted by U.
When set has no elements—it is called the empty set or null set.

Set operations—
Union—it is set of all elements which belong to A or to B.
Intersection—it is set of all elements which belong to both A and B.

If two sets have no common elements—they referred to disjoint or non–intersecting. If S is union of two set A & B and there are no common elements in A & B—in such cases S is called as ‘disjoint union’ of A and B.
